## camp machine list

List configured machines

### Synopsis

List every machine in ~/.obey/machines.yaml, plus the implicit "local" machine
(this machine, never persisted to the file).

```
camp machine list [flags]
```

### Options

```
  -h, --help   help for list
      --json   Output as JSON
```
